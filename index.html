Sobre Mim

Desenvolvedor Full Stack

Sou um desenvolvedor apaixonado por tecnologia, com experiência em desenvolvimento mobile, backend e web. Domino Flutter para criação de aplicativos eficientes e intuitivos, C# e Python para desenvolvimento backend escalável e seguro, além de JavaScript e React para interfaces web dinâmicas e responsivas. Tenho habilidades em criação de APIs, bancos de dados, automações e segurança de aplicações, sempre aplicando boas práticas e arquiteturas robustas. Meu foco é desenvolver soluções inovadoras, escaláveis e bem estruturadas, garantindo boa performance, usabilidade e integração entre sistemas. Além disso, busco constantemente aprimorar meus conhecimentos em inteligência artificial, ciência de dados e IoT 5G, aplicando essas tecnologias para criar soluções inteligentes e estratégicas.
